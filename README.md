# Supply Chain Risk Monitor

Supply Chain Risk Monitor is a machine learning dashboard that predicts the impact of global supply chain disruptions. Three Random Forest models — trained on 58 historical events spanning 2001 to 2025 — forecast disruption severity (medium / high / extreme), expected freight rate shock, and estimated recovery time given a user-defined scenario. Features include disruption type, affected region, duration, and structural flags such as strait closures and port shutdowns. The dashboard layers these predictions onto five years of weekly port congestion data, 25 years of shipping rate history, and annual industry vulnerability scores across ten sectors, giving analysts a single view of both real-time market signals and forward-looking risk estimates.

Link to Dashboard via Streamlit: **https://supply-chain-project-9exjq5ke7cxza6w2trnxgz.streamlit.app/**

## Features

| Page | Description |
|---|---|
| **Risk Predictor** | Enter a disruption scenario → get ML predictions for severity, freight shock %, and recovery months |
| **Disruption History** | Browse and filter 58 major events; scatter, bar, and detail table views |
| **Industry Vulnerability** | Exposure heatmap for 10 industries; JIT vs inventory buffer; nearshoring progress |
| **Shipping Markets** | Baltic Dry Index, container spot rates, supply chain pressure index, on-time delivery |
| **Port Congestion** | Weekly congestion index, wait days, and vessels-at-anchor for 14 major ports |

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the ML models (takes ~10 seconds)
python train.py

# 3. Launch the dashboard
streamlit run app.py
```

## Data

Five CSV files covering 2000–2025:

| File | Description |
|---|---|
| `disruption_events.csv` | 58 major disruptions with impact metrics (freight shock, GDP impact, recovery time) |
| `industry_exposure.csv` | Annual vulnerability scores for 10 industries across 8 risk dimensions |
| `port_congestion.csv` | Weekly congestion data for 14 major global ports (2019–present) |
| `shipping_rates.csv` | Monthly Baltic Dry Index, container rates, air cargo, and pressure index |
| `trade_flows.csv` | Bilateral trade flows across key corridors with tariff rates and concentration risk |

## Models

Three Random Forest models trained on historical disruption events:

| Model | Target | Notes |
|---|---|---|
| Severity Classifier | medium / high / extreme | Stratified 5-fold CV, balanced class weights |
| Freight Shock Regressor | freight rate shock (%) | 5-fold CV R² reported at training |
| Recovery Regressor | recovery time (months) | 5-fold CV R² reported at training |

> **Note:** The training set contains 58 events. CV scores reflect this limited sample — treat predictions as directional estimates, not precise forecasts.
